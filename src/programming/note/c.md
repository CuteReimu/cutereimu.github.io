### 一、前言，基本算法，数据结构，标准输入输出语句

1. 什么叫常量，什么叫变量，变量的声明：

   ```c
   int i = 0;
   ```


2. 强制类型转换：

   ```c
   char c = (char) i;
   ```

3. 什么叫表达式，`while` 循环语句，`if ... else` 条件语句
4. `printf();`与`scanf();`

### 二、数组

1. 数组的声明

   ```c
   int a[3] = {1, 2, 3};
   ```

2. 字符数组，字符串，数组名代表数组第一个元素的地址

   ```c
   char str[] = "hello world";
   char *p = str;
   ```

3. 二维数组

   ```c
   int a[3][3];
   ```

### 三、函数

1. 函数的声明

   ```c
   int sum(int a, int b, int c）｛
       return a + b + c;
   }
   ```

2. 函数的参数与返回值
3. 函数的形式参数与实际参数，函数参数的传递
4. 数组名作函数的参数
5. 函数的递归调用

### 四、指针

1. 变量声明

   ```c
   int *p;
   ```

   表示指向`int`的指针类型

2. 运算符`*`与`&`，指针指向数组元素，指针变量的加减运算
3. 指向指针的指针，运算符`*`的多重运用
4. 指针数组的声明

   ```c
   int *a[10];
   ```

   表示指向`int`的指针构成的数组

5. 指向行向量的指针，指针与二维数组的运用，运算符`*`与加减的混合运用，一维数组名代表数组第一个元素的地址
6. 指针函数的声明

   ```c :no-
   int a;
   int *f() {
       return &a;
   }
   ```

   其中返回值的数据类型为`int *`，表示指向`int`的指针类型

7. 指向函数的指针
8. 指针作函数的参数
9. `malloc()`函数
